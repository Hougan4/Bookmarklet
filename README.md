# Bookmarklet
よく使う自作ブックマークレット


使い方：Google Chromeのブックマークを編集し、URL欄にhoge.jsの中身を貼り付ける

#

### Scrapbox-Bookmarklet
- WebサイトのURLを"[サイトのタイトル␣URL]" の形式でコピーできる
- [Scrapbox](https://scrapbox.io/)にWebサイトのURLを貼り付ける際に便利
